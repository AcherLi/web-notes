## Path 模块
Node.js path 模块提供了一些用于处理文件路径的小工具，我们可以通过以下方式引入该模块：
```javascript 
const path = require('path')
```
### 常用API
| 方法                               | 作用                                |
| ---------------------------------- | ---------------------------------- |
| path.normalize(p)                  | 规范化路径，注意'..' 和 '.'。        |
